# Demand for Electric Bicycles

## Project Overview

**Yulu** is India’s leading micro-mobility service provider, offering shared electric bicycles for daily commutes. Yulu aims to eliminate traffic congestion in India by providing safe, affordable, solo, and sustainable commuting solutions through a user-friendly mobile app.

Yulu zones are located at appropriate places such as metro stations, bus stands, office spaces, residential areas, and corporate offices to make first-mile and last-mile travel smooth, affordable, and convenient.

Recently, Yulu has suffered considerable dips in revenue. The company has contracted a consulting company to understand the factors on which the demand for shared electric bicycles depends, specifically in the Indian market.

## Business Problem

Yulu wants to understand the key factors affecting the demand for shared electric bicycles in India.

The company wants to answer the following questions:

1. Which variables are significant in predicting the demand for shared electric bicycles in the Indian market?
2. How well do those variables describe electric bicycle demand?

## Dataset Description

The dataset file `bike_sharing.csv` contains hourly aggregated usage data with the following columns:

| Column       | Description                                                              |
| :----------- | :----------------------------------------------------------------------- |
| `datetime`   | Beginning of the hour for which the data is aggregated                   |
| `season`     | Season of the year                                                       |
| `holiday`    | Whether the day is a holiday or not                                      |
| `workingday` | Whether the day is neither a weekend nor a holiday                       |
| `weather`    | Weather condition                                                        |
| `temp`       | Air temperature in degrees Celsius                                       |
| `atemp`      | Feeling temperature in degrees Celsius                                   |
| `humidity`   | Air humidity in percent                                                  |
| `windspeed`  | Wind speed in km/h                                                       |
| `casual`     | Number of bikes rented by casual users                                   |
| `registered` | Number of bikes rented by registered users                               |
| `count`      | Total number of rented bikes, including both casual and registered users |

## Feature Details

### Season

| Value | Season |
| :---- | :----- |
| 1     | Spring |
| 2     | Summer |
| 3     | Fall   |
| 4     | Winter |

### Weather

| Value | Weather Condition                                                                       |
| :---- | :-------------------------------------------------------------------------------------- |
| 1     | Clear, few clouds, scattered clouds                                                     |
| 2     | Mist + cloudy, mist + scattered clouds, mist + few clouds, mist                         |
| 3     | Light snow, light rain + thunderstorm + scattered clouds, light rain + scattered clouds |
| 4     | Heavy rain + ice pallets + thunderstorm + mist, snow + fog                              |

### Working Day

| Value | Description                            |
| :---- | :------------------------------------- |
| 1     | Day is neither a weekend nor a holiday |
| 0     | Day is a weekend or holiday            |

* Relevant tables and visualizations

* Interpretation of how well those variables explain electric bicycle demand
