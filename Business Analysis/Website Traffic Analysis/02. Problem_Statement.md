# Website Traffic Analysis

## Project Overview

**Linkfire:** Analyze web traffic patterns using click-rate metrics and event distributions.

**Technologies:** Python, Pandas, SciPy, Matplotlib

## Assignment

The goal of this project is to better understand website traffic, particularly the volume and distribution of events, and to develop ideas for increasing link click rates.

Analyze the data using Python and the specified libraries, providing answers to the following questions:

1. **[Pandas]** How many total pageview events did the links in the provided dataset receive during the full period? How many pageviews did they receive per day?
2. **[Pandas]** What about the other recorded event types?
3. **[Pandas]** Which countries did the pageviews come from?
4. **[Pandas]** What was the overall click rate, calculated as `clicks / pageviews`?
5. **[Pandas]** How are click rates distributed across different links?
6. **[Pandas & SciPy]** Is there any correlation between clicks and previews for a link?

   * Is the correlation statistically significant?
   * How large is the effect?
   * Test for both potential linear and categorical relationships, including binary relationships, between the two variables.

## Data Description

The provided dataset, `traffic.csv`, contains web traffic data represented as events from several pages, referred to as links.

The data covers a period of seven days and includes categorical dimensions describing:

* The geographic origin of the traffic
* The content associated with each page, represented by `isrc`
